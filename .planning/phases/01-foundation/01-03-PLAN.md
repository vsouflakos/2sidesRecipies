    Create the Unit model + migration: `php artisan make:model Unit -m --no-interaction`.

    In the units migration `up()`, define:

    ```php
    Schema::create('units', function (Blueprint $table) {
        $table->id();
        $table->string('name');                       // e.g. 'gram'
        $table->string('symbol');                      // e.g. 'g'
        $table->string('type');                        // 'weight' | 'volume' | 'count'
        $table->decimal('base_factor', 16, 6)->nullable(); // grams or ml per unit; null for count units
        $table->timestamps();
        $table->unique(['name']);
    });
    ```

    In `app/Models/Unit.php`, set `protected $fillable = ['name', 'symbol', 'type', 'base_factor'];`
    and a `casts()` method returning `['base_factor' => 'decimal:6']`.

    Create the Allergen model + migration: `php artisan make:model Allergen -m --no-interaction`.

    In the allergens migration `up()`, define:

    ```php
    Schema::create('allergens', function (Blueprint $table) {
        $table->id();
        $table->string('name');                        // e.g. 'Gluten'
        $table->string('slug')->unique();              // e.g. 'gluten'
        $table->text('note')->nullable();              // optional EU sub-detail
        $table->timestamps();
    });
    ```

    In `app/Models/Allergen.php`, set `protected $fillable = ['name', 'slug', 'note'];`.

    Run `php artisan migrate`, then `vendor/bin/pint --dirty --format agent`.
