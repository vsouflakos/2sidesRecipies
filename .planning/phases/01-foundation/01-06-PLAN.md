    Implement `handle()` to resolve the locale from the authenticated user, fall back to a session-stored

    ```php
    <?php

    namespace App\Http\Middleware;

    use Closure;
    use Illuminate\Http\Request;
    use Illuminate\Support\Facades\App;
    use Symfony\Component\HttpFoundation\Response;

    class SetLocale
    {
        private const SUPPORTED = ['en', 'el'];

        public function handle(Request $request, Closure $next): Response
        {
            $locale = $request->user()?->locale
                ?? $request->session()->get('locale', 'en');

            App::setLocale(in_array($locale, self::SUPPORTED, true) ? $locale : 'en');

            return $next($request);
        }
    }
    ```

    Implement `__invoke`:

    ```php
    public function __invoke(UpdateLocaleRequest $request): RedirectResponse
    {
        $locale = $request->validated('locale');

        if ($request->user()) {

        }

        $request->session()->put('locale', $locale);

        return back();
    }
    ```
