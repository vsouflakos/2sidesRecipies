<!-- Plan 04 routes + UI-SPEC version contracts. -->

Plan 04 routes (Wayfinder TS — import from @/actions / @/routes):
- POST /recipes/{recipe}/versions (recipes.versions.store) — partial reload; body { change_note? }
- POST /recipes/{recipe}/draft/recall (recipes.draft.recall) — partial reload; body { expected_sequence }; returns 409 on sequence mismatch
- GET /recipes/{recipe}/versions/compare?a=&b= (recipes.versions.compare) — renders recipes/versions/compare
- POST /recipes/{recipe}/duplicate (recipes.duplicate) — redirect to new recipe show
- PUT /recipes/{recipe}/draft (recipes.draft.update) — used to update a sub-recipe pin (one logical edit)

show.tsx props include `versions` (list: {id, version_number, committed_at, change_note, is_current}) and `recipe.sections[].lines[]` where a sub-recipe line has `sub_recipe_version_id` set and the resource carries `pinned_version_number` + `latest_version_number` of the component.

UI-SPEC contracts (authoritative): sub-recipe line variant = name + pinned `Badge v{N}` (secondary) + gram qty `Input` + accent "Update available" `Badge` when newer exists + delete; clicking the cue opens a `Dialog` "Update sub-recipe pin?" with body "Update '{name}' from {vOld} to {vNew}? This will change the metrics for this recipe." and buttons "Update to {vNew}" / "Keep {vOld}". Save Version = `Button variant="default"` "Save Version" opening a `Dialog` for an optional ≤140-char "What changed?" note with a "Save without note" secondary; success sonner "Version {vN} saved." Recall = `Button variant="outline"` "Recall", no confirm, disabled (aria-disabled, tooltip "Nothing to undo") when the edit log is empty; sends expected_sequence; 409 → sonner "Draft updated elsewhere — please refresh." Version history = `Sheet side="right"` 320px, rows "v{N} · {date} · {note or —}" with a "Current" badge and a "Compare" button; selecting two opens the compare view. Compare view = two columns, row-level diffs highlighted `bg-yellow-50` / `bg-yellow-900/20` PLUS a diff icon (color not sole signal — accessibility). All copy in the UI-SPEC Copywriting Contract — exact EN, add EL.

Circular reference (RECIPE-06): the draft endpoint returns a 422 when a picked sub-recipe would cycle; surface inline below the combobox row: "Cannot add '{name}' — it would create a circular reference."
