UI-SPEC contracts (authoritative): panel is a sticky `div` `bg-muted rounded-lg p-4`, scrolls independently; nutrition uses a `Switch` toggle "Per portion"/"Per 100 g"; cost section has a currency-prefixed selling-price `Input`, food cost % recomputes live; allergen "contains" = `Badge` `text-destructive`, "may contain" = `Badge` `text-muted-foreground`, named not icon-only; baker's section CONDITIONAL on a flour-base line existing; scaling controls = scale-factor `Input` + portions `Input`, changes view-only + italic quantities, "Apply to Draft" button appears when differing; data gap banner = `Alert variant="destructive"` outline listing the missing ingredient names. Mobile: panel collapses to a bottom summary bar (cost/portion + kcal/portion) expanding to a bottom `Sheet`. All copy in UI-SPEC Copywriting Contract — exact EN strings, add EL.

selling_price persistence (RESEARCH Open Q3): persisted into recipe_drafts.data — changing it triggers an auto-save (one logical edit). Scaling/portions are view-only (RECIPE-07/08): client-side recompute ONLY, no server round-trip, until "Apply to Draft".
