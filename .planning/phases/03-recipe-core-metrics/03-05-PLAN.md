Build the single-page recipe builder: the create page, the builder shell (`show.tsx`), named section blocks containing ingredient lines and steps, the unified inline search combobox, the quick-create-ingredient modal, the metadata block, and the 600 ms debounced auto-save hook.

Purpose: The builder is where the chef builds the recipe ‚Äî the living document. It posts every edit to the Plan 04 draft endpoint. Per CONTEXT.md it is the working draft: there is no explicit save-draft step.

</objective>
