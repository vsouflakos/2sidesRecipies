RecipeTestResource serialized shape (from plan 04-02):
```
id, recipe_id, recipe_version_id, version_number, type ('trial'|'experiment'),

ratings ([{dimension, score, is_custom}] | null), hypothesis, outcome_narrative,

photos ([{id, path, url, order}]), created_at, updated_at
```

POST recipes.tests.store / PUT recipes.tests.update accept (multipart):
```
type, recipe_version_id, tested_at, overall_rating, tasting_notes,
ratings[] {dimension, score, is_custom}, hypothesis, outcome_narrative, verdict,
change_rows[] {what_changed, expected_effect, actual_effect}, photos[] (File),
deleted_photo_ids[] (update only)
```
